# godoymods

## Portfolio Simples com Abas e Filtro

Projeto de portfolio que apresenta projetos em abas separadas e permite filtrá-los por tags. Um exemplo interativo de HTML, CSS e JavaScript.

## Habilidades Demonstradas

- **HTML Semântico:** Criei uma estrutura HTML bem organizada e semântica para o conteúdo da página.

- **CSS Flexbox:** Utilizei Flexbox para criar layouts flexíveis e responsivos, garantindo a aparência desejada em diferentes tamanhos de tela.

- **CSS Estilização:** Ganhei experiência na aplicação de estilos CSS para melhorar a aparência do site, incluindo formatação de texto, layout responsivo e uso de flexbox para alinhamento.

- **JavaScript DOM Manipulation:** Manipulei o DOM com JavaScript para criar abas interativas e funcionalidade de filtro.

- **JavaScript Interatividade:** Aprofundei meus conhecimentos em JavaScript, aprendendo como manipular o DOM para criar abas interativas e funcionalidades de filtro.

- **Lógica de Programação:** Implementei lógica para alternar entre abas e filtrar projetos com base nas tags.

- **Versionamento Git:** Utilizei Git para controle de versão, permitindo o rastreamento de alterações e colaboração eficaz.

- **Design Responsivo:** Criei estilos responsivos que se adaptam a diferentes dispositivos e tamanhos de tela.

## Como Usar

1. Clone ou faça o download deste repositório.

2. Abra o arquivo `index.html` no seu navegador para visualizar o portfolio, ou simplesmente [clique aqui.](https://rafaelgodoyebert.github.io/godoymods/)

## Recursos Utilizados

- HTML: Estrutura da página.
- CSS: Estilos da página e responsividade.
- JavaScript: Lógica de exibição das abas e filtro dos projetos.

## Personalização e Expansão

- **Adicionar Novos Projetos:** Adicione novos projetos no arquivo `index.html` seguindo o formato das seções existentes.

- **Editar Tags:** Personalize as tags de cada projeto no atributo `data-tags` na div `project-tags`.

- **Personalizar Estilos:** Ajuste os estilos da página no arquivo `styles.css`.

- **Modificar Comportamento:** Personalize o comportamento das abas, filtro ou outras funcionalidades no arquivo `script.js`.
